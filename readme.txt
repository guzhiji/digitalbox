tools.zip 提供的工具：
1.
install.php
系统安装
帮您完成本系统的安装工作
2.
setmaster.php
更改站长
帮您解决忘记密码的问题
3.
synccache.php
同步缓存
帮您解决本地设置缓存被损坏或意外被更改的问题
4.
syncuploadedfiles.php
同步上传文件
帮您解决本地上传文件与数据库记录不一致的问题

*tools.zip 提供的所有工具，均会对网站管理构成安全上的威胁
 当您使用完后，您可以通过 toolchecker.php ，来删除它们

*toolchecker.php 不需删除，它不会有安全隐患，
 反而能更好地帮助您管理好 tools.zip 提供的所有工具

*您应该将 tools.zip 保留好，以备再次使用
